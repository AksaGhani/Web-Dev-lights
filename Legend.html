How To Play

The aim of the game is to turn on the right coloured light bulbs using the coloured lightbulbs presented, matching the randomly coloured boxes presented to the right

  1. First press the button in order to produce a light sequence
  2. The light-bulbs with ID from 4-6 will all turn on, each with a different colour
  3. You must remember the sequence of colours that was produced
  4. The player must then try to switch all of the lights off by clicking the correct coloured light-bulb on the screen
  5. The player does this by clicking on the light-bulb to turn. If the colour clicked does not match the original colour sequence it will change the colour of the light-bulb. But if the correct colour is clicked, the light will turn off.
  6. Once the player thinks they have turned off all the correct lights, they must click the button
  7. If the player was successful at the game, all of the lights will turn green.
  8. In order to refresh the game or start again, the player must click
  9. Then press Go for the light sequence

These Light-bulbs act as buttons which the user should use to turn the light that colour.

In the Game page you'll notice that the light-bulbs have been put into two different columns. Each column represents the light-bulbs in SMB109, where the 1st light-bulb has the ID:1 and the last light-bulb in the column has the light ID:3. This is the same for both columns.

The light sequence will only be using the lights with ID 4-6, therefore, the player only has a choice of one of two colours for each light bulb. For example, the light sequence will show: blue (light ID 4) , orange (light ID 5), purple (light ID 6). For light ID 1 the player has the option to either choose blue or red. This is the same for the remaining two lights.
