How To Play

The aim of the game is to turn on the right coloured light bulbs using the coloured lightbulbs presented, matching the randomly coloured boxes presented to the right

  1. First press the button in order to produce a light sequence (these will only be shown once and cannot be seen after the first light-bulb has been turned on by the player)
  2. The light-bulbs with ID from 1-3 will all turn on, each with a different colour
  3. You must remember the sequence of colours that was produced
  4. The player must then recreate the colour sequence using the light-bulbs provided in the application
  5. The player does this by clicking on the light-bulb to turn it on
  6. Once the player thinks they have matched all the lights, click the button

These Light-bulbs act as buttons which the user should use to turn the light that colour.

In the Game page you'll notice that the light-bulbs have been put into two different columns. Each column represents the light-bulbs in SMB109, where the 1st light-bulb has the ID:1 and the last light-bulb in the column has the light ID:3. This is the same for both columns.

The light sequence will only be using the lights with ID 1-3, therefore, the player only has a choice of one of two colours for each light bulb. For example, the light sequence will show: blue (light ID 1) , orange (light ID 2), purple (light ID 3). For light ID 1 the player has the option to either choose blue or red. This is the same for the remaining two lights.
